# 機能リスト

## 基本機能 (MVP)

- **タスク管理 (CRUD)**
    - タスクの作成
    - タスクの一覧表示
    - タスクの詳細表示
    - タスクの更新
    - タスクの削除
    - 見積工数の設定
- **タスクの依存関係**
    - あるタスクがどのタスクに依存しているかを設定できる。
    - あるタスクがどのタスクから依存されているかを表示できる。
- **ステータス管理**
    - タスクの状態（例：未着手, 進行中, 完了）を管理できる。
- **期限設定**
    - タスクに期限日を設定できる。

## 拡張機能

- **ユーザー認証**
    - ユーザー登録・ログイン機能。
    - タスクをユーザーごとに管理する。
- **依存関係の可視化**
    - タスク間の依存関係をグラフなどで視覚的に表示する。
- **通知機能**
    - 期限が近づいたタスクを通知する。
- **検索・フィルタリング**
    - キーワードやステータス、期限でタスクを絞り込み表示する。
- **コメント機能**
    - 各タスクにコメントを追加できる。
